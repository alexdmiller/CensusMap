Info

The code for this project is available on GitHub.

To receive support or report bugs, use Twitter (@admiller), email (alexmiller@gmail.com) or file an issue on GitHub.

US Census data is provided by the census.gov API. The FCC Census block conversion API is used to find Census tracts numbers based on longitude and latitude coordinates.

This site uses the American Community Survey 2007-2011, which means the numbers shown are estimates, not actual census counts. To learn about the accuracy of ACS data, read this document. None of the charts on this site display error bars, which should scare you and cause you to question the validity of this site. Please don't make policy decisions using this tool.

According to the Census Bureau, Hispanics or Latinos can be of any race, any ancestry, any ethnicity, or any country of origin. Therefore the inclusion of the Hispanics/Latinos in the "Racial Distribution" chart is slightly misleading.
