Census data provided by the US Census.gov API. Currently, this site is using the ACS 2007-2011 API, which means the numbers shown are estimates, not actual census counts. To learn about the accuracy of ACS data, read this document. None of the charts on this site display error bars, which should scare you and cause you to question the developer's integrity. Please don't make policy decisions using this site.

According to the Census Bureau, Hispanics or Latinos can be of any race, any ancestry, any ethnicity, or any country of origin. Therefore the inclusion of the Hispanics/Latinos in the "Racial Distribution" chart is slightly misleading. However, race and ethnicity are arbitrary, socially constructed concepts anyway.
